# Contributing to Really Nicca

Thank you for your interest in contributing!

## Getting Started

1. Read the [Development Framework](./Development_Framework.md) to understand our workflow.
2. Review the [Code Consistency Standards](./CODE_CONSISTENCY.md) for coding guidelines.
3. Check [TASKS.md](../TASKS.md) for current progress and open tasks.

## Pull Request Process

1. Create a feature branch from `develop`.
2. Ensure all tests pass.
3. Update documentation if necessary.
4. Submit a Pull Request.

## Coding Standards

We enforce strict coding standards for logging, linting, and type safety.
Please run `npm run lint` and `npm run format` before committing.
